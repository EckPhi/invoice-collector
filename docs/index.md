# Invoice Collector Documentation

Welcome to the Invoice Collector documentation! Invoice Collector is a powerful, free, and open-source Docker application designed to automatically retrieve invoices and receipts from various suppliers.

<p align="center">
  <img src="https://github.com/invoice-collector/invoice-collector/raw/master/images/preview/preview.svg" width="100%" />
</p>

## What is Invoice Collector?

Invoice Collector seamlessly connects to customer portals, APIs, and email inboxes to automatically gather invoices from your suppliers. It eliminates the tedious task of manually downloading invoices from multiple sources, saving you time and reducing errors.

## Key Features

✨ **Automated Collection** - Automatically retrieves invoices from multiple sources
🔐 **Secure Storage** - Stores credentials securely using Bitwarden integration
🐳 **Docker-Based** - Easy deployment with Docker and Docker Compose
📊 **MongoDB Integration** - Reliable data storage with MongoDB
🌐 **REST API** - Complete API for integration with your systems
🔌 **Extensible** - Easy to add new collectors for additional suppliers
🌍 **Multi-language Support** - Available in multiple languages

## Supported Suppliers

Invoice Collector comes with built-in support for major suppliers including:

- **Amazon** - E-commerce platform
- **Bureau Vallée** - Office supplies
- **Carrefour** - Retail supermarket
- **Free** - Internet service provider
- **Intermarché** - Retail supermarket
- **Leclerc** - Retail supermarket
- **Leroy Merlin** - Home improvement
- **OpenAI** - AI services
- **OVH** - Cloud services
- **U** - Retail supermarket

And many more! The community is constantly adding new collectors.

## Use Cases

- **Small Businesses** - Automate invoice collection for accounting
- **Accountants** - Centralize client invoice collection
- **Personal Finance** - Track expenses automatically
- **Enterprise** - Integrate with existing financial systems
